СПИСОК ИНСТРУМЕНТОВ

1. GetCategories
   Категория: Инструменты каталога услуг
   Описание: Получение списка категорий услуг

2. GetServices
   Категория: Инструменты каталога услуг
   Описание: Получение списка услуг по категории

3. ViewService
   Категория: Инструменты каталога услуг
   Описание: Просмотр детальной информации об услуге

4. FindSlots
   Категория: Инструменты бронирования
   Описание: Поиск слотов по временным признакам (утро, вечер и т.д.)

6. CreateBooking
   Категория: Инструменты бронирования
   Описание: Создание новой записи/бронирования

7. FindMasterByService
   Категория: Инструменты бронирования
   Описание: Поиск мастера по услуге

8. GetClientRecords
   Категория: Инструменты для работы с записями клиентов
   Описание: Получение списка записей клиента

9. CancelBooking
   Категория: Инструменты для работы с записями клиентов
   Описание: Отмена существующей записи

10. RescheduleBooking
    Категория: Инструменты для работы с записями клиентов
    Описание: Перенос записи на другое время

11. CallManager
    Категория: Инструмент передачи менеджеру
    Описание: Передача диалога менеджеру

12. AboutSalon
    Категория: Инструмент информации о салоне
    Описание: Получение информации о салоне

13. Masters
    Категория: Инструмент информации о мастерах
    Описание: Получение информации о мастерах
